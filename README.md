# CRM-Sales-performance-Analysis-

This project I conducted a analysis on closed-won CRM sales data from 2017 to evaluate sales team performance, identify underperforming agents, analyze quarter revenue  trends, and product performance throughout the year. 
It uses imported MySQL views and insights visualizations using Power BI 

#Dataset
- Source: https://mavenanalytics.io/data-playground/crm-sales-opportunities
- Description: The dataset contains sales and customer data, including information about agents, products, sales dates, revenue.

#Tools Used
-MySQL: Querying, cleaning, and aggregating data 
-Power BI: Interactive dashboards for insights and reporting 

#Analysis
1. Imported the dataset into MySQL and cleaned the data as needed.
2. Created SQL queries to calculate total sales, profits, product revenue, and revenue by teams.
3. Built Power BI dashboards to visualize:
   -Sales Performance by manager teams and region
   -Quarter-over-quarter revenue trends
   -Product-level insights
   -Total of unperformance agents per team manager and region
